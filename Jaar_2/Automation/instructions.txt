Python http.server - officiële documentatie

http.server can also be invoked directly using the -m switch of the interpreter. This serves files relative to the current directory:

python -m http.server

The server listens to port 8000 by default. The default can be overridden by passing the desired port number as an argument:

python -m http.server 9000

By default, the server binds itself to all interfaces.

By default, the server uses the current directory. The option -d/--directory specifies a directory to which it should serve the files. For example, the following command uses a specific directory:

python -m http.server --directory /tmp/

Nota: de by default geinstalleerde python interpreter van RHEL9 is /usr/bin/python3

Examen indienen

Vergeet je screenshots niet.

Je kan het meegeleverde script ./package_exam.sh gebruiken vanuit de hoofddirectory. 

Hier zijn de manuele stappen als het script voor problemen zou zorgen:

Stel je naam is Leroy Jenkins, dan ga je 1 directory hoger staan dan je voornaam_achternaam/ hoofddirectory en gebruikt dit commando om een .tgz file aan te maken:

tar -czvf leroy_jenkins.tgz --exclude='*.vagrant' --exclude='*documentation' leroy_jenkins/

Deze file upload je naar BlackBoard.

Nadat alles gelukt is, sluit je het examen op BlackBoard.
SHA-256 hash berekenen

Je kan het meegeleverde script ./package_exam.sh gebruiken vanuit de hoofddirectory.

Zo bereken je de sha-256 hash:

sha256sum leroy_jenkins.tgz

Vul van deze hash value de eerste 4 karakters en de laatste 4 karakters in op je examenvoorblad.
