Hoe ziet er het examen uit?

 

Theorie; het theorie examen bestaat uit 60 multiple choice vragen, waarbij duidelijk gespecifieerd wordt per vraag wanneer er meer dan 1 antwoord kan aangeduid worden.

Het examen bevat vragen uit alle delen van de leerstof, en kan zowel praktisch als theoretisch gericht zijn. Er zal geen zeer specifieke praktijkvragen gesteld worden, maar IP instellingen, verwijzingen of duidelijke weetvragen kunnen wel gesteld worden. Alle vragen komen uit de powerpoints of uit de leerstof die duidelijk aangeduid is tijdens de lessen (bv VPN, routing, etc).

 

Prakijk: het praktijk examen is een Word file die vanaf de ftp server kan gedownload worden. In het Word document staan een aantal opdrachten. Deze opdrachten bevatten instellingen. Deze instellingen moet je configureren in je vm en screenshotten, waarna je ze toevoegt in het Word document. Aparte screenshots worden NIET verbeterd, enkel het Word document. In het Word document staat specifiek over welke instellingen gescreenshot moeten worden. Je zorgt zelf voor correcte screenshot software. Windows heeft bv een knip tool (snipping tool) ter beschikking. Je bent zelf verantwoordelijk voor het maken van screenshots en het bezitten van goede software. Je Word file is een docx file, ook hier ben je verantwoordelijk dat je deze bestanden kan lezen en ook opslaan.

 

Er is enkel een server ter beschikking, een client mag je (maar is zeker niet noodzakelijk) zelf installeren en toevoegen om instellingen te testen, maar dit is NIET noodzakelijk. Dat is volledig aan jou (ook toevoegen aan het domein is je eigen verantwoordelijkheid). Let op de ip instellingen op de server, deze kunnen verkeerd staan door import. Je mag het interne ipadres 192.168.1.1/24 gebruiken. Enkel de instellingen op de server (en dus de screenshot) worden verbeterd. Het toevoegen, installeren en/of aanpassen van de client zijn NIET nodig voor het afleggen van het examen.
